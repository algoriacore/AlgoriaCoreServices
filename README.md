# AlgoriaCoreServices
Proyecto de servicios REST .Net Core

Es una aplicación de servicios REST y gRPC basada en .Net Core.
Puede correr sobre Windows, Linux y cualquier otra plataforma que soporte .Net Core.

Implementa toda la lógica del lado del servidor y realiza el procesamiento y persistencia de datos.
Está desarrollada en C# y puede ser consumida por cualquier aplicación cliente que soporte servicios REST.
